# decrypter
A simple python decrypter for encrypted messages

# Get started:
```
# set up virtual env
virtualenv venv

# activate virtual env
. venv/bin/activate

# install requirements on virtual env
pip install -r requirements.txt
```

# Exit virtual env:
```
deactivate
```

# Run development server:
```
export FLASK_APP=app.py && flask run
```
